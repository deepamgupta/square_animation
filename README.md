# square_animation
Play with a big red square

- A simple app with a square popping up in the begining.
- You can play with this square.
- You can count :
	- Number of taps.
	- Number of Double taps.
	- Number of Long presses.
- Movement :
	- Move the square up.
	- move the square down.
